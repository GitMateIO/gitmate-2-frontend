Security

Server Security

GitMate is hosted at DigitalOcean which is secured with 24/7/365 onsite security, security cameras as well as biometric readers and at least two factor authentication.

To read more about the physical server security visit DigitalOcean Security.

Employee Access

GitMate employees will never access your source code, analysis results, or any information associated to your user account unless you grant your explicit consent to do so, e.g. for support cases.

Reporting Security Issues

If you think you found a security issue, please create a confidential issue at code.gitmate.io or mail it directly to incoming+gitmate/open-source/gitmate-2@gitlab.com .
